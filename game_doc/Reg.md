# Reg
描述游戏内的注册信息。
## 静态函数

### Reg.ItemID

```
static int ItemID(string name)
```

根据物品名返回注册的动态ID。

### Reg.ItemIDName

```
static string ItemIDName(int id)
```

根据动态ID返回物品的全局ID名称。

### Reg.MaxItemID

```
static int MaxItemID()
```

返回最大物品ID。
