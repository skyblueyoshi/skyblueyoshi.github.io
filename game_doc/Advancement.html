Advancement

Advancement

描述一项成就。

属性

成员 类型 描述
itemID int 成就使用的物品ID，用于在成就树界面显示物品图标。
parentID int 成就在成就树的父成就ID。
showTip boolean 玩家完成该成就时，是否在界面右上角弹出完成成就提示。
broadcast boolean 玩家完成成就时，是否广播给所有人。
