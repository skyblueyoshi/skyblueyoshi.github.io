Vector2

描述一个2维向量。

属性

成员 类型 描述
x number
y number
length number 向量长度。
lengthSquared number 向量长度的平方。
angle number 向量到原点的角度。
zero Vector2 返回零向量。
one Vector2 返回全1向量。
unitX Vector2 返回X轴单位向量。
unitY Vector2 返回Y轴单位向量。
